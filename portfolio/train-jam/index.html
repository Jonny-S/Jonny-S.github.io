Train Jam!

A first person cooperative game inspired by the Train Jam. Currently a WIP, it is built in Unreal Engine 4 with logic created using Blueprints.

Train Interaction

At this early stage I have been responsible for building out the gameworld and creating logic for animating train objects. The train models communicate with a ‘timetable’ created by colleagues to trigger an arrival or departure. This game logic is modular allowing it to be interfaced with by other blueprints.
