Project Coffee Table

Work in progress experimental game. Based on the themes of tranquility, cooperation and simplicity.

Using realistic water simulation provided by NVIDIA Flex technology, the player will use non-traditional input to guide the water around each plane of an architectural stack. When the player reaches the goal on each table a new table will replace it creating perpetual motion and randomly shuffled game spaces.
